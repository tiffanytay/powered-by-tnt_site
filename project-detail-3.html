Project Overview

In partnership with the Data Changemakers, No Kill Colorado--a nonprofit animal welfare advocacy organization--wished to transform the vast, complex world of animal shelter data into a compelling story that could motivate action and reform.

As part of this effort, I developed solutions that turned complicated, multi-year datasets from over 300 shelters and rescues into clear, actionable insights.

My work shines a light on opportunities for improvement, empowering the public, policymakers, and the media to understand and engage with the current landscape of animal welfare in Colorado. By bridging technical solutions with advocacy, the project not only increased transparency but also strengthens No Kill Colorado's position as a trusted leader driving positive change for pets statewide.

The Challenge

  • Inconsistent data formatting across 21 annual shelter statistics files spanning from 2007 to 2024.
  • Lack of standardized unique facility identifiers, complicating analysis.
  • Integration of statistics, regulatory facility list and, if available, updated location data.

The Solution

Solution Approach

I unified and cleaned complex shelter and nonprofit data using Python and Jupyter, then built interactive Power BI and Tableau dashboards to visualize key animal welfare metrics and facilitate actionable insights.

Opportunities Dashboard in Power BI

Maps in Tableau

  • Implemented a comprehensive, automated data wrangling pipeline in Jupyter Notebook to unify and clean disparate datasets.
  • Built an interactive Power BI dashboard with alert features to identify opportunities for improving animal welfare in Colorado.
  • Created two Tableau Public maps: one allowing exploration of various key statistics by region, and another visualizing Live Release Rate (LRR) with coloring to highlight low-performing regions.
  • Designed Tableau worksheets enabling direct searching and data quality spot-checks.
